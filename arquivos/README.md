# Arquivos

Esta pasta contém os textos processados e scripts de podcast gerados pelo projeto Na Boca do Sol.

## Autores

- [Jorge Amado](jorge_amado/) - Escritor brasileiro, militante comunista
- [Mao Zedong](mao_zedong/) - Líder revolucionário chinês

## Estrutura

Cada autor possui uma pasta com:
- `README.md` - Biografia e lista de obras
- Subpastas para cada texto processado contendo:
  - `Titulo.txt` - Texto extraído
  - `Titulo_podcast.txt` - Script do episódio

## Fonte

Todos os textos são de domínio público, disponibilizados pelo [Marxists Internet Archive](https://www.marxists.org/).
